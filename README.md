# CorreBia!
Jogo simples desenvolvido em Python, com a biblioteca Pygame, inspirado no tradicional e mundialmente conhecido Pacman.
Esse é o projeto final da matéria de Sistemas Operacionais, que possuia como principal objetivo ver na prática o uso de multi Threads. 
No jogo, cada fantasma roda em uma Thread distinta e possui um algoritmo de perseguição próprio.
